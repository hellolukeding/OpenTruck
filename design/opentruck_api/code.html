POST /v1/chat/completions

Chat Completions

Creates a model response for the given chat conversation. Use this endpoint to generate natural language text based on a sequence of messages between a user and an AI model.

Request Body

model string REQUIRED

ID of the model to use. You can use gpt-4o, claude-3-5-sonnet, or our optimized opentruck-v2.

messages array REQUIRED

A list of messages comprising the conversation so far.

role string

The role of the messages author. One of system, user, assistant, or tool.

content string

The contents of the message.

temperature number Defaults to 1

What sampling temperature to use, between 0 and 2. Higher values like 0.8 will make the output more random, while lower values like 0.2 will make it more focused and deterministic.

stream boolean Defaults to false

If set, partial message deltas will be sent, as in ChatGPT. Tokens will be sent as data-only server-sent events as they become available.

Example Request 
curl https://api.opentruck.ai/v1/chat/completions \
  -H "Content-Type: application/json" \
  -H "Authorization: Bearer $OPENTRUCK_API_KEY" \
  -d {
    "model": "gpt-4o",
    "messages": [
      {
        "role": "system",
        "content": "You are a helpful assistant."
      },
      {
        "role": "user",
        "content": "Hello!"
      }
    ]
  }
